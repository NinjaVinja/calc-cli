# 🖩 calc-cli

A terminal-based calculator in Python with a stylish ASCII art interface — built as part of the **100 Days of Code** Python bootcamp journey.

---

## ✨ Features

- ASCII art logo with a classic calculator display
- Supports four operations: **Addition**, **Subtraction**, **Multiplication**, **Division**
- **Chain calculations** — continue using the previous result as your next first number
- **Restart** the calculator fresh at any time
- Clean, minimal CLI interaction

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x

### Run

```bash
git clone https://github.com/NinjaVinja/calc-cli.git
cd calc-cli
python calculator.py
```

---

## 🕹️ Usage

```
Please enter your first number: 10
+
-
*
/
Pick an operation: *
Please enter your second number: 5
10.0 * 5.0 = 50.0
Type 'y' to continue calculating with 50.0, or type 'n' to start a new calculation:
```

- Type **`y`** → chain the result into the next calculation
- Type **`n`** → restart fresh
